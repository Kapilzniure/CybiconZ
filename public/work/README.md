# Work Image Assets

This folder contains image assets used for the portfolio and case studies.

## Required files

- `/public/work/lwangblack/hero.jpg` — main hero image for the LwangBlack Coffee case study
  - Recommended size: `1200x675`
- `/public/work/lwangblack/detail-1.jpg` — supporting product/detail shot
  - Recommended size: `800x600`
- `/public/work/johnnies/hero.jpg` — main hero image for the Johnnies Liquor case study
  - Recommended size: `1200x675`
- `/public/work/johnnies/detail-1.jpg` — supporting product/detail shot
  - Recommended size: `800x600`

## Image guidance

- Use dark, premium photography or styled product shots.
- Keep the tone consistent with a Tokyo digital agency brand:
  - clean composition
  - strong contrast
  - minimal clutter
  - subtle brand accents where appropriate

## Fallback behavior

Until real photos are available, the site will show a branded placeholder using the project name and service color. When the local image path 404s, it falls back to a generated placeholder component.
